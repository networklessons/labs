# Packet Captures

This folder contains packet capture files of different network protocols. These capture files can be opened and analyzed with tools like [Wireshark](https://www.wireshark.org/).

The captures include examples of:

- Routing protocols (BGP, OSPF, EIGRP, IS-IS, RIP)
- VPN technologies (DMVPN, IPsec, GRE)
- MPLS and Traffic Engineering
- VXLAN and EVPN
- Spanning Tree Protocol
- Multicast protocols (IGMP, PIM)
- IPv6 protocols
- And many more network protocols
