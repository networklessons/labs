# VIRL Labs

These are legacy labs for [Cisco VIRL 1.x](https://learningnetworkstore.cisco.com/cisco-modeling-labs-personal/cisco-cml-personal). 

VIRL has been replaced by [Cisco CML (Cisco Modeling Labs)](https://www.cisco.com/c/en/us/products/cloud-systems-management/modeling-labs/index.html). These labs can be imported into Cisco CML.
